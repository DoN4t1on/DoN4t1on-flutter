## Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK

### Installation

1. Clone the repository:

```bash
git clone https://github.com/DoN4t1on/DoN4t1on-flutter.git
```

2. Navigate into the project directory:

```bash
cd DoN4t1on-flutter
```

3. Get dependencies:

```bash
flutter pub get
```
### Development

To run the app in development mode, use:

```bash
flutter run
```
